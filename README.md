---

## 🚀 Demonstração Visual

### 📽️ Hero Cinematográfico & App Showcase
O site conta com um background de vídeo local em 4K (Navio) para impacto imediato e uma seção de demonstração do aplicativo com som ativado, eliminando as objeções de qualidade do cliente.

### 💰 Comparativo de Economia & Planos
Integramos uma calculadora de economia visual (R$ 222,50 vs R$ 30,00) que torna a decisão de compra racional e irresistível.

### 🛡️ Segurança & Confiança (Facebook Ads Ready)
Rodapé estruturado com Políticas de Privacidade, Termos de Uso e Disclaimers obrigatórios para evitar bloqueios em campanhas de tráfego pago.

---

## 🔥 Diferenciais Técnicos

- **Funil de Vendas AIDA**: Estrutura psicológica que guia o cliente da Atenção ao Fechamento.
- **Pulse CTAs**: Todos os botões de conversão possuem animações constantes de "pulsação" para atrair cliques.
- **Scroll Reveal**: Efeito de surgimento suave (1.2s) de todas as seções durante a navegação.
- **Zero Buffering**: Uso de vídeos locais em vez de YouTube para carregamento instantâneo.
- **100% Responsivo**: Layout otimizado para celulares, visando o tráfego mobile do Instagram/Facebook.
- **FAQ Inteligente**: Seção de dúvidas frequentes para quebrar as principais objeções de venda automaticamente.

## 🏁 Como Iniciar o Projeto

1. **Clonar o Repositório**:
   ```bash
   git clone https://github.com/SEU-USUARIO/smart-hub-tv.git
   ```

2. **Instalar Dependências**:
   ```bash
   npm install
   ```

3. **Rodar em Desenvolvimento**:
   ```bash
   npm run dev
   ```

4. **Configurar o Pixel do Facebook**:
   Edite o arquivo `src/app/layout.tsx` e substitua `SEU_ID_DO_PIXEL` pelo ID da sua conta de anúncios.

---

## 🚢 Deploy (Lançamento)

Recomendamos o deploy via **Vercel** para máxima performance:

1. Crie uma conta na [Vercel](https://vercel.com).
2. Conecte seu repositório do GitHub.
3. Clique em **Deploy**. O projeto será otimizado e lançado automaticamente com Certificado SSL grátis (cadeado verde).
